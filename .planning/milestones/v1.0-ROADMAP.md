# Milestone v1.0: GSD Cursor Integration

**Status:** ✅ SHIPPED 2026-02-05
**Phases:** 1-3
**Total Plans:** 5

## Overview

Adding Cursor IDE support to the main GSD installer by consolidating the conversion logic from cursor-gsd/. This enables users to install GSD directly to ~/.cursor/ using `npx get-shit-done-cc --cursor --global`.

## Phases

### Phase 1: Core Implementation

**Goal**: User can run `npx get-shit-done-cc --cursor --global` and GSD installs correctly to `~/.cursor/` with all files properly converted for Cursor format.
**Depends on**: None (first phase)
**Plans**: 2 plans

Plans:
- [x] 01-01-PLAN.md — Conversion functions (tool mapping, frontmatter, body text)
- [x] 01-02-PLAN.md — Installer integration (CLI, directory handling, install/uninstall)

**Requirements:**
- CONV-01..05: Tool name mapping, conversion function, frontmatter conversion, color conversion, body text replacement
- PATH-01..04: Directory names, global dir, path replacement, command format
- INST-01..05: CLI flag, interactive prompt, install/uninstall, attribution
