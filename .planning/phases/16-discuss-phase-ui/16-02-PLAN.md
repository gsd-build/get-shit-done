Create `contextParser.ts`:
- Export Decision interface: { id: string, content: string, locked: boolean, section: 'decisions' | 'specifics' | 'deferred', isNew?: boolean }
- Export ContextMdState interface: { domain: string, decisions: Decision[], specifics: Decision[], deferred: Decision[], raw: string }
- Export parseContextMd(markdown: string): ContextMdState
  - Parse XML-like sections: &lt;domain&gt;, &lt;decisions&gt;, &lt;specifics&gt;, &lt;deferred&gt;
  - Extract bullet points from each section as decisions
  - Generate stable IDs from content hash (for tracking changes)
  - Default locked: false for all decisions

Create `contextSerializer.ts`:
- Export serializeContextMd(state: ContextMdState): string
  - Reconstruct CONTEXT.md from state
  - Preserve locked status as metadata comment (<!-- locked -->)
  - Maintain original formatting

Create `contextStore.ts`:
- State: phaseId, contextState (ContextMdState), previousState (for diff), lastUpdated
- Actions:
  - setContext(markdown: string) - parse and set, mark new items
  - toggleLock(decisionId: string) - toggle single decision
  - bulkLock(section: string, locked: boolean) - lock/unlock all in section
  - updateDecision(id: string, content: string) - for editing
  - getChangedDecisions() - returns decisions where isNew is true
  - clearNewFlags() - called after highlight animation completes
