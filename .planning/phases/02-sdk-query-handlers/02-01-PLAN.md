
**Handler 2: smeDetectProcesses (SDK-02)**
- Signature: `export const smeDetectProcesses: QueryHandler = async (args, projectDir, workstream) => { ... }`
- Same config guard pattern (return `{ data: { enabled: false, matches: [] } }` when disabled or loadConfig throws)
- Parse args: extract `--file-paths` values (all args between `--file-paths` and next flag or end), extract `--goal` value (the arg after `--goal`)
- Same readdir pattern for smes dir (return empty matches on catch)
- Filter to `-SME.md` files, extract frontmatter for each
- For each SME document, check if `process_name` (lowercased) appears in:
  - Any file path (lowercased) via `filePath.toLowerCase().includes(processName.toLowerCase())`
  - The goal string (lowercased) via `goal.toLowerCase().includes(processName.toLowerCase())`
- Track match source: `'file-path'`, `'keyword'`, or `'both'`
- Deduplicate: one match entry per process, with `match_source` reflecting all match types
- Return `{ data: { enabled: true, matches } }` where each match is `{ file, process_name, block_mode, match_source }`

**Handler 3: smeContextBlock (SDK-03)**
- Signature: `export const smeContextBlock: QueryHandler = async (args, projectDir, workstream) => { ... }`
- Same config guard pattern (return `{ data: { enabled: false, found: false, process: args[0] ?? '', block: '' } }` when disabled)

- Construct smes dir, readdir, find the file matching `{processName}-SME.md` (case-insensitive match on the process_name part)

- If found: readFile the SME document, extract frontmatter for metadata

**Arg parsing helper (private function within sme.ts):**
```typescript
function parseDetectArgs(args: string[]): { filePaths: string[]; goal: string } {
  const filePaths: string[] = [];
  let goal = '';
  let i = 0;
  while (i < args.length) {
    if (args[i] === '--file-paths') {
      i++;
      while (i < args.length && !args[i]!.startsWith('--')) {
        filePaths.push(args[i]!);
        i++;
      }
    } else if (args[i] === '--goal') {
      i++;
      if (i < args.length) {
        goal = args[i]!;
        i++;
      }
    } else {
      i++;
    }
  }
  return { filePaths, goal };
}
```
