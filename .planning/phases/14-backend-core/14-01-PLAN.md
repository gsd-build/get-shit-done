   - Export ApiEnvelope<T> interface per CONTEXT.md: { data: T, meta: { timestamp: string, requestId: string }, error?: { code: string, message: string, details?: unknown } }
   - Export envelopeMiddleware that:
     - Generates requestId via crypto.randomUUID()
     - Sets requestId in context via c.set('requestId', requestId)
     - Wraps JSON responses in envelope format after next()
   - Export success() helper: (c, data, status?) => envelope JSON response
   - Export error() helper: (c, code, message, status, details?) => envelope JSON with error field
