# Todo App

A simple todo application for demonstrating GSD Dashboard features.

## Overview

This is a sample project used to demonstrate the GSD (Get Shit Done) workflow and dashboard capabilities.

## Tech Stack

- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: SQLite

## Status

**Current Milestone**: v1.0 - Basic Todo Features
**Current Phase**: Phase 2 - Task Management
