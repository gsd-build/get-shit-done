# GSD Plan Phase

Create an execution plan for a phase with task breakdown and dependency analysis.

## Usage

```
/gsd-plan-phase $1
```

## Creates

- `.planning/phases/XX-name/XX-YY-PLAN.md` — execution plans

## Process

Execute the plan-phase workflow from `get-shit-done/workflows/plan-phase.md`.

See `commands/gsd/plan-phase.md` in the GSD source for the full command definition.
