# GSD Map Codebase

Map and analyze the codebase structure.

## Usage

```
/gsd:map-codebase [focus-area]
```

**Focus areas:** tech, arch, quality, concerns

## Process

This command spawns gsd-codebase-mapper agents to analyze the codebase.
