# GSD Next

Detect the current project state and automatically invoke the next logical GSD workflow step.

## Usage

```
/gsd:next
```

## Process

Execute the next workflow from `get-shit-done/workflows/next.md` end-to-end.

## Reference

See `commands/gsd/next.md` in the GSD source for the full command definition.
