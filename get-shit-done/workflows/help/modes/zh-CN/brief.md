**GSD — 核心命令速览**

```text
/gsd:new-project           初始化项目（全新项目）
/gsd:map-codebase          映射现有代码库（棕地项目）
/gsd:plan-phase <N>        创建阶段计划
/gsd:execute-phase <N>     执行阶段
/gsd:progress              我在哪、下一步是什么
/gsd:quick                 小型临时任务，保留 GSD 保障
/gsd:fast "<task>"         简单内联任务 — 无子代理
/gsd:debug "<symptom>"     持久调试会话（跨 /clear 保持）
/gsd:capture               保存想法 / 待办 / 笔记
/gsd:ship <N>              从已完成阶段创建 PR
```

更多信息：`/gsd:help`（默认导览）· `/gsd:help --full`（完整参考）· `/gsd:help <topic>`（单章节）
</reference>
