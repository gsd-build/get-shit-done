### Column Definitions

- **Control ID:** Unique identifier (e.g., CC6.1-01, A1.2-01)
- **Control Description:** What the control does
- **TSC Criteria:** Trust Service Criteria addressed (e.g., CC6.1, CC6.2)
- **Category:** Security / Availability / Processing Integrity / Confidentiality / Privacy
- **Type:** Preventive / Detective / Corrective
- **Frequency:** Annual / Quarterly / Monthly / Weekly / Daily / Per Occurrence / Continuous
- **Risk Level:** High / Moderate / Low
- **Test Method:** Inspect / Reperform / Observe / Inquiry / Combination
- **Status:** Not Tested / In Progress / Effective / Exception / N/A
- **Exceptions:** Count of exceptions found (0 if none)
- **Work Paper Ref:** Reference to work paper in .audit/workpapers/{Control-ID}/
